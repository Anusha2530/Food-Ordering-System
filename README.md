# Food Ordering System 🍔

## 📌 Project Overview
The Food Ordering System is a full-stack web application developed using PHP, MySQL, HTML, CSS, and JavaScript.  
This project allows users to register, login, browse food items, add products to cart, place orders, and manage their accounts.  
An admin panel is also included for managing products, users, and orders.

# ▶️ How to Run the Project

1. Install XAMPP

2. Move project folder into:
C:\xampp\htdocs\

3. Start Apache & MySQL from XAMPP

4. Create database in phpMyAdmin:
food_ordering_system

5. Import/Create tables

6. Run project in browser:
http://localhost/food-ordering-system
